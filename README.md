# CPP_IMAGE_PROC

This repository contains C++ programs demonstrating basic image processing operations using OpenCV.

## Repository structure

- `code/` - C++ source files for various image processing tasks.
- `main_img/` - Example input images used by the programs.
- `outputs/` - Output images generated by the code.

## Included programs

- `canny_edge.cpp` - Canny edge detection.
- `gaussian_median.cpp` - Gaussian blur and median filtering.
- `hist_equal.cpp` - Histogram equalization.
- `img_color_conv.cpp` - Color space conversion.
- `img_read_disp.cpp` - Image loading and display example.
- `laplacian_hpf.cpp` - Laplacian high-pass filter.
- `region_growing.cpp` - Region growing segmentation.
- `thresh_img.cpp` - Image thresholding.

## Usage

Compile a program with g++ and link against OpenCV. Example:

```bash
g++ code/canny_edge.cpp -o canny_edge `pkg-config --cflags --libs opencv4`
./canny_edge
```

Adjust the compile command depending on your OpenCV installation and version.
